## 驗證
- npm install：通過
- npm run build：通過

## 使用方式
1. 解壓縮本專案。
2. 用 VS Code 開啟資料夾。
3. 執行 `npm install`。
4. 執行 `npm run dev`。
5. 測試結果頁與 PDF：點「列印 / 另存 PDF 報告」，在瀏覽器列印視窗選擇「另存為 PDF」。
